<p>This is a modern Flappy Bird game developed using HTML5, CSS3, and JavaScript. The game is rendered on the HTML5 Canvas and includes smooth animations, responsive controls, and attractive visual effects. Players control a bird by jumping through randomly generated pipes while avoiding collisions to achieve the highest possible score.</p>

<h1>Features</h1>
<p>HTML5 Canvas-based game rendering</p>
<p>Smooth bird movement with gravity and jump physics</p>
<p>Randomly generated colorful pipes</p>
<p>Dynamic animated background with stars, clouds, and mountains</p>
<p>Particle effects on scoring and jumping</p>
<p>Real-time score and best score tracking</p>
<p>Responsive design for desktop and mobile devices</p>
<p>Touch controls for mobile users</p>
<p>Keyboard controls (Space, Arrow Up, Enter)</p>
<p>Game Over screen with medal system</p>
<p>Modern Glassmorphism UI design</p>
