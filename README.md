<p>This is a modern Flappy Bird game developed using HTML5, CSS3, and JavaScript. The game is rendered on the HTML5 Canvas and includes smooth animations, responsive controls, and attractive visual effects. Players control a bird by jumping through randomly generated pipes while avoiding collisions to achieve the highest possible score.</p>

<h1>Features</h1>
HTML5 Canvas-based game rendering
Smooth bird movement with gravity and jump physics
Randomly generated colorful pipes
Dynamic animated background with stars, clouds, and mountains
Particle effects on scoring and jumping
Real-time score and best score tracking
Responsive design for desktop and mobile devices
Touch controls for mobile users
Keyboard controls (Space, Arrow Up, Enter)
Game Over screen with medal system
Modern Glassmorphism UI design
